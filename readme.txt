Experimental plugin to build forms.

== Description ==

Create form in FrMk menu, then create a page and insert form's shortcode.

== Installation ==

1. Download the latest version of the plugin.
1. Extract the download to your websites wp-content/plugins folder.
1. Activate the plugin from the WordPress Admin plugins section.
